## Laptop Verify
Access all relevent laptop specifications at a glance under one simple UI, no need to scour through multiple complicated UI and command lines. Verify whether a laptop contains aftermarket / counterfeit parts to assist in buying and authenticating second hand laptops.



This repo contains the front end code for the app. The authentication check is powered by Google's Gemini LLM.

Note: The backend of the app is deployed on Render.com.
